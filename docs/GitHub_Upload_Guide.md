# GitHub Upload Guide

## 1. Create a GitHub Repository

Recommended repository name:

```text
brain-tumor-mri-classification
```

Description:

```text
Educational brain MRI classification project using ResNet18 transfer learning.
```

Choose:

- Public, if you want to show it in your portfolio
- Private, if you only want to keep it for yourself first

## 2. Upload Files

Upload the whole folder:

```text
brain-tumor-mri-classification-project
```

Important files to keep:

- `README.md`
- `requirements.txt`
- `notebooks/Brain_Tumor_Detection_Colab_Report.ipynb`
- `docs/Code_Explanation_Guide.md`
- `slides/Brain_Tumor_MRI_Classification_Intern_Report.pptx`
